# DistilledAI Lab

Public AI experiment lab for Distilled Science.

## Routes

- `/` - The Lab landing page and experiment cabinet.
- `/playground` - Cognitive Bias Playground with six interactive demos and localStorage progress.
- `/prompt` - The exact build prompt, preserved verbatim.

## Stack

- Next.js App Router
- Tailwind CSS
- Motion for React
- Static routes, no backend, no env vars
- Self-hosted Geist font files

## Commands

```bash
npm run dev
npm run lint
npm run build
npm run og
```

The deployed production domain is `https://test.distilledai.org`.
